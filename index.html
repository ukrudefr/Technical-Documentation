Kopfbereiche auszeichnen

Das Element <header> kann laut HTML-Spezifikation eine Überschrift, einleitende Inhalte oder auch Navigationshilfen enthalten und ist für einen solchen Kopfbereich ideal geeignet.

Das Element <header> kann auf einer Seite mehrfach vorhanden sein, und dieses kann beim Gestalten per CSS zu unerwünschten Überraschungen führen, zum Beispiel die folgende CSS-Regel:

header p {
margin: 0;
}
 Der Selektor header p gilt für alle Header-Elemente auf einer Seite, und er wählt alle darin enthaltenen Absätze aus.

Während die Namen der Elemente wie <header>, <nav>, <main> und <footer> Teil der HTML-Spezifikation sind, sind die Namen der Klassen relativ frei wählbar:

<header class="site-header">
Navi erstellen

Eine Navigation ist eine Sammlung von Hyperlinks, die zu anderen Inhalten der Webseite führt. Das nav- Element ist für wichtige Navigationsblöcke auf einer Seite reserviert und sollte nicht für jede x-beliebige Linkliste genutzt werden.

Genau wie header kann auch nav auf einer Seite mehrfach auftreten, und daher ist es eine gute Idee, den Navigationsbereichen mit einer Klasse einen zusätzlichen Namen mit auf den Weg zu geben.

Die Navigation ist auf jeder Seite ähnlich, und daher beginnt der Name der Klasse wie beim Kopfbereich mit der Vorsilbe site-. Mit diesen Klassen im HTML können Sie die Selektoren im Stylesheet ändern und die Elemente genauer auswählen:

  • nav wird zu .site-nav
  • nav ul wird zu .site-nav ul
  • nav li wird zu .site-nav li
  • nav a wird zu .site-nav a

Navigationsbereiche können nicht nur in und um einen Kopfbereich auftauchen. Sehr beliebt ist z.B eine Navigation für rechtliche Pflichtlinks im Footer.

Der Hauptinhalt einer Webseite

Mit dem Element main hat der Hauptinhalt einer Seite bildlichgesprochen auch einen eigenen Stuhl und kann es sich so auch mal bequem machen.

Das Element main hat folgende Charakteristika:

  • main markiert den Hauptinhalt einer Seite.
  • main darf auf jeder Seite nur ein einziges Mal vorkommen.
  • main enthält normalerweise auf jeder Seite anderen Inhalt.

Das Element main kann auch eine Klasse erhalten namens site-content. Außerdem spendieren Sie dem Inhaltsbereich noch eine ID, die Sie z.B. content nennen:

<main class="site-content" id="content">
<h2>Wilkommen</h2>
</main>
Fußbereiche auszeichnen

Im Fußbereich einer Webseite finden sich häufig Informationen über die Webseite. In HTML gibt es dazu das Element <footer>.

Außer einem Link zurück nach oben stehen im Footer häufig Links zu Impressum und Datenschutzrichtlinien, aber auch andere Inhalte wie Social-Media-Links werden dort gerne untergebracht.

<footer>
<a href="#top">Nach oben> </footer>

Das Element <footer> kann genau wie <header> mehrfach auf einer Seite auftauchen und auch innerhalb von Abschnitten oder Artikeln enthalten sein.

Inhaltliche Abschnitte erstellen

Für die inhaltlihe Gliederung einer Webseite oder einzelnen Teilen davon gibt es das Element <section<, auf Deutsch Abschnitt. Das section-Element bezeichnet einen Bereiche einer Webseite oder eine thematische Gruppierung von Inhalten und sollte mit einer Überschrift anfangen.

Zum Kennenlernen von <section> unterteilen Sie den Inhaltsbereich der Beispielseite in zwei Abschnitte:

  • Die bereits vorhandene Einleitung mit der Überschrift Willkommen, zwie Absätzen und einer Liste soll von section-Element mit der Klasse content-intro umgeben.
  • Danach folgt ein zweiter Abschnitt mit der Klasse infoboxen, der mit h2- Überschrift anfängt und im nächsten Abschnitt mit drei article-Elementen gefüllt wird.

Im HTML sieht es so aus:

<main class="site-content">
<section class="content-intro">
<h2>Willkommen</h2>

Eigenständige Blöcke

Das Element article dient genau wie section zur Gruppierung von Elementen und zur Unterteilung der Inhalte in Abschnitte.

Unterschiede zwischen den beiden:

  • Ein Artikel ist anders als ein einfacher Abschnitt und stellt in sich eine geschlossene komposition dar.
  • Ein Artikel sollte für sich allein stehen können und auch außerhalb der Webseite, auf der er veröffentlicht wurde, verständlich sein.
Bereiche mit zusätzlichen Informationen

Das Element aside ist für Inhalte gedacht, die den umgebenden Text erläutern oder nur marginal mit ihm in Verbindung stehen.

Das Element aside kann z.B. für Seitenleisten verwendet werden, die auf großen Bildschirmen häufig rechts oder links neben dem Hauptinhalt stehen.

Andere Beispiele für die Verwendung von aside wären:

  • Blöcke mit Werbung
  • Linklisten, die keine Navi-Bereiche sind
  • Inhalte, die nicht direkt zum Hauptinhalt der Seite gehören
Elemente semantisch neutral gruppieren

Das div-Element ist ein Container, eine Kiste, eine Box, in der mehrere andere Kisten aufbewahrt werden können, um sie gemeinsam zu gestalten. Das ist bei der täglichen Arbeit an Webseiten oft sehr praktisch. Es dient zur Gruppierung von Elementen, wenn von den semantischen Strukturelemente keines passt
